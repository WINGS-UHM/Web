excerpt: "5G and emerging Next-G systems improve performance, reliability, and security through higher data rates, low-latency communication, massive device support, and stronger authentication. However, false base station (FBS) attacks remain a persistent threat because user equipment still relies on broadcast signaling and early pre-authentication procedures before secure contexts are fully established. This work explores FBS threats in 5G with a focus on core network interaction, extending the threat model beyond traditional UE-side spoofing, downgrade, and disconnection attacks. In particular, we examine how a fake gNB can interact with the 5G core through the N2 interface and NGAP protocol."
